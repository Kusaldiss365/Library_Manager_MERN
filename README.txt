Library Management System - MERN Stack

This system allows you to Insert, edit, delete, update and search for books as a library management system.

Technologies:
I have used React with Mui, NodeJS, expressJS and MongoDB technologies.

Setup:

Backend -
1. Install packages -
	npm install express mongoose
	npm i express-async-handler
	npm i dotenv

2. To Run: node server.js

Frontend - 
1. Install packages -
	npm install

2. To Run: npm start

Additional Notes:
Empty inputs are not accepted at data entry windows
